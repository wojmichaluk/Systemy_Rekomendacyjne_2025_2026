# Systemy_Rekomendacyjne_2025_2026
### Repository for Systemy Rekomendacyjne (Recommendation Systems) course at AGH University

Repozytorium zawiera rozwiązania zadań z kolejnych laboratoriów (jupyter notebooki **.ipynb**) oraz wszystkie dodatkowe, wykorzystywane w ramach laboratorium pliki - katalogi **lab*i***, `i = 1, 2,..., 6`.
W laboratorium nr 3 pliki **.csv** w podkatalogach `socialmediadata-beeradvocate/data` oraz `socialmediadata-beeradvocate/original` zostały skompresowane (**.zip**) i w takiej postaci umieszczone ze względu na ich zbyt duży rozmiar, ale w pozostałych przypadkach pliki pomocnicze są niezmienione.

Zamieszczone tutaj rozwiązania zadań odpowiednio przedstawione podczas oceny pozwoliły zgarnąć pełną pulę punktów z laboratoriów.
Oczywiście mogą one zawierać błędy, które spowodowały np. niestandardowe wartości badanej metryki, ale nacisk był położony na odpowiednie podejście do zadania i wykorzystanie wiedzy, a nie doskonałość implementacyjną.
Przede wszystkim mam nadzieję, że okażą się przydatne. Korzystajcie rozważnie!
